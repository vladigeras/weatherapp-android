## 📱 About

Weatherapp provides current weather information for any location using API. The app is modern
Android development with:

- **Kotlin** + **Coroutines** for asynchronous work
- **Jetpack Compose** for declarative UI
- **Hilt** for dependency injection
- **MVVM** architecture with a single‑activity setup
- **Material Design 3** theming
- Unit tests with **JUnit**, **MockK**, and **Turbine**

## ⚙️ Requirements

- **Minimum Android SDK**: 35 (Android 15)
- **Java**: 17

---

## 🔒 Privacy

App contains **no advertisements**, **does not collect** any personal or usage data, and **does not
transmit** any information from your device. The app works entirely offline except for fetching
weather data from the public API, which only receives the location coordinates you explicitly
provide. We never sell, share, or monetize your data, and we have no plans to introduce ads or
tracking in the future.

---

## 📦 Usage

- Launch the app – it will request location permission.
- Grant permission to see weather for your current location.
- Use the search bar to look up any city worldwide.
- Pull‑to‑refresh to fetch the latest data.
